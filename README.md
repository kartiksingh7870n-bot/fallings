# fallings

एक सरल डेटिंग वेब ऐप जो प्रोटोटाइप प्रोफाइल, स्वाइप जैसी लाइक/डिसलाइक क्रियाएँ, फ़िल्टर और मैच सूची प्रदान करता है।

## कैसे चलाएं

1. `index.html` ब्राउज़र में खोलें, या
2. टर्मिनल में:
   - `cd /workspaces/fallings`
   - `python3 -m http.server 8000`
   - फिर `http://localhost:8000` खोलें।

## फ़ाइलें

- `index.html` — मुख्य यूआई
- `styles.css` — ऐप स्टाइल
- `app.js` — प्रोफाइल स्वाइप और मैच लॉजिक
- `package.json` — सरल स्टार्ट स्क्रिप्ट
